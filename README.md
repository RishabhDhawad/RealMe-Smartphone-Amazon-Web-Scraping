# 📱 Realme Smartphone Price Scraper

A Jupyter Notebook project that scrapes the latest **Realme smartphone titles and prices** from [Amazon India](https://www.amazon.in/), and exports the data into a neatly organized Excel file.

![Jupyter](https://img.shields.io/badge/Jupyter-Notebook-orange.svg)
![Web Scraping](https://img.shields.io/badge/Web--Scraping-Beautifully--Done-success.svg)

---

## 📌 Project Overview

This project uses **AutoScraper** to automatically extract:
- 📦 Product titles
- 💰 Prices

From Amazon India listings of Realme smartphones, and saves them in an Excel file using **pandas**.

---

## 🚀 Features

- 🔍 Scrapes Realme phone names and their prices from Amazon.
- 📄 Automatically structures and stores the data in an Excel file (`.xlsx`).
- 💡 Easy-to-understand and beginner-friendly Jupyter Notebook.
- 🧠 Built using intelligent rule learning via AutoScraper.

---

## 🛠️ Tech Stack

- **Python**
- **Jupyter Notebook**
- **AutoScraper**
- **pandas**
- **openpyxl**
